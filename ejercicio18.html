Pestañas dinamicas

Para hacer que las pestañas sean conmutables, agregue el atributo data-toggle = "tab" a cada enlace. Luego agregue una case .tab-pane con un ID único para cada pestaña y envuélvalas dentro de un elemento div con la clase .tab-content.
