Listas desplegables

La clase .dropdown se usa para indicar un menú despegable.

Use la clase .dropdown-menu para contruir el menú despegable.

Para abrir el menú despegable, use un botón o un enlace con una clase de .dropdown-toggle y data-toggle = "dropdown".
